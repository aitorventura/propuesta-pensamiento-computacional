# Propuestas de Formación para el CEFIRE

## 📚 Cursos propuestos

### 🐍 [Introducción a la programación en Python](curso-python.md)

**Dirigido a:** docentes de Informática (FP y Bachillerato) — **40 horas · 7 módulos**

Curso de Python desde cero, con ejercicios relacionados con el contexto de los ciclos formativos. Se trabajan variables, condicionales, bucles, listas, funciones, manejo de ficheros y automatización del sistema, y se cierra con una práctica propia elegida por cada participante.

→ [Módulos y contenidos del curso](curso-python.md)

### 🤖 [Herramientas IA para crear contenido didáctico](curso-ia-docente.md)

**Dirigido a:** cualquier docente de Secundaria, Bachillerato o FP — **40 horas · 7 módulos**

Uso práctico de **Gemini**, **ChatGPT**, **NotebookLM**, **Napkin** y **PartyRock** para generar materiales, rúbricas, esquemas y recursos interactivos. Incluye un módulo para publicar apuntes en la web con Markdown, Google Antigravity y GitHub Pages. Sin conocimientos previos de informática.

→ [Módulos y contenidos del curso](curso-ia-docente.md)
